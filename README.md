# First Circle Pay Dashboard 
A dashboard that showcases transaction and batch processing with Analytics and Batch Processing Templates. 

## Installation 

```js
npm install

```

##  Running the app locally

```js

npm run dev

```

## Accessing the app locally
```js

localhost:5173

```

## Running the Storybook

```js

npm run storybook

```

## Accessing the Storybook
```js

localhost:6006

```

# Design Process
- Branding, I focused on creating a dashboard with Transactions and Analytics for Batch Processing using the design elements from First Circle Pay website. 
- The logo for the app is named First Circle Pay, keeping the accent color of the brand. 
- Accent Colors applied thru all the design components that showcase a unified UI/UX design specifically for First Circle Pay.

# Technical Decisions
- Tech stack that were used is React.js + Vite + Storybook + TypeScript + Radix UI + TailwindCSS
- Third-party libraries were used for other components
- UI/UX were designed with Figma, no prior UX research were done but kept in mind the branding for First Circle

# Areas of Improvement
- As this exercise was only done on a 3-day period there are areas of improvement to make this app scalable and can be used for production
- Additional Unit and UI/UX Testing should be added to ensure production-readiness
- Additional monitoring tools should be added
- Refining of the UI/UX Design
